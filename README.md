An interactive Healthcare Analytics Dashboard built using Power BI, Power Query, and DAX to analyze hospital operations, patient insights, financial performance, and operational metrics through interactive visualizations.
# 🏥 Healthcare Analytics Dashboard

## 📌 Project Overview

The Healthcare Analytics Dashboard is an interactive Business Intelligence solution developed using Power BI. It helps analyze hospital data by transforming raw information into meaningful insights for decision-making. The dashboard focuses on hospital operations, patient demographics, clinical performance, financial performance, and operational efficiency.

This project demonstrates end-to-end dashboard development, including data cleaning, data transformation, DAX calculations, data modeling, and interactive report design.

---

## 🎯 Project Objectives

- Analyze hospital performance using interactive dashboards.
- Monitor patient admissions and discharge trends.
- Evaluate clinical performance across departments.
- Analyze financial performance and revenue trends.
- Improve operational efficiency using data-driven insights.
- Enable users to filter and explore data dynamically.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dashboard Pages

### 🏥 Hospital Overview
Provides a summary of hospital performance, patient statistics, department analysis, admissions, discharges, and overall KPIs.

### 🩺 Clinical Analysis
Analyzes patient diagnoses, disease severity, gender distribution, doctor specialization, and clinical trends.

### 💰 Financial Analysis
Displays revenue, insurance claims, payment methods, billing analysis, and financial KPIs.

### ⚙️ Operational Analysis
Monitors hospital operations, patient flow, department efficiency, and operational performance metrics.
